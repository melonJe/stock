# 📈 주식 자동매매 프로그램

기업의 fundamental 분석을 통해 매수할 주식을 선별하고, 주가 분석 기반의 알고리즘을 활용하여 적절한 매매 타이밍에 증권사에 자동으로 매매를 요청하는 Python 기반의 자동화 시스템입니다.


## 🚀 프로젝트 목적

- 수동으로 반복되는 주식 분석과 매매 과정을 자동화하여 시간 절약 및 투자 효율 극대화
- 투자자의 전략을 반영한 매수/매도 로직 구현 및 테스트

  
## 🧩 주요 기능

- 📊 **기업 Fundamental 분석**: 재무제표 기반으로 유망 종목 선별 및 북마크 기능 제공
- ⏰ **예약 매매 시스템**: 주가 조건 만족 시 자동으로 한국투자증권 API를 통해 매매 실행
- 🔄 **주기적 데이터 갱신 및 분석**: 주식 시장 실시간 모니터링 및 자동 리밸런싱 기능 (예정)
